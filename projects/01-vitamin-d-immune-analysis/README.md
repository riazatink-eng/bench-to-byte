# Vitamin D and Immune Biomarker Analysis

## Project Overview

This project reconstructs statistical analysis workflows from my doctoral research examining associations between vitamin D status and circulating immune biomarkers in a human cohort study.

The goal of this project is to present the analytical reasoning, SAS/R workflow structure, statistical methods, and interpretation of publication-style biomedical data analysis.

This project emphasizes annotated code, exported results, workflow documentation, and scientific interpretation.

## Research Objective

To evaluate whether vitamin D status is associated with inflammatory and immune activation markers in clinically healthy adults.

## Dataset

The original analysis was based on a human cohort study with biomarker, demographic, clinical, and immune marker data.   

## Methods

The analytical workflow includes:

- data cleaning and variable preparation
- descriptive statistics
- biomarker distribution assessment
- covariate selection
- multivariable regression modeling
- nonlinear or threshold-based modeling where appropriate
- multiple testing correction
- interpretation of immune biomarker patterns

## SAS and R Workflow
 
Primary statistical analyses were conducted in SAS.

R may be used for selected visualizations, exploratory plots, and figure reconstruction.

## SAS Skills Demonstrated

- data import and cleaning
- variable recoding
- cohort subsetting
- descriptive statistics
- regression modeling
- model adjustment
- ODS output export
- table generation
- statistical reporting

## SAS Procedures Used

- PROC IMPORT

- ODS HTML

## Key Outputs
