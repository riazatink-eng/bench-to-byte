# Project Summary

## Project Title

Vitamin D Status and Immune Biomarker Patterns in a Human Cohort Study

## Background

Vitamin D has been linked to immune regulation, but the relationship between vitamin D status and immune biomarker patterns in generally healthy adults is complex.

This project reconstructs the analytical framework from my doctoral research examining associations between vitamin D status and circulating immune biomarkers.

## Research Question

Is vitamin D status associated with inflammatory and immune activation markers in a human cohort after accounting for demographic and clinical covariates?

## Analytical Goal

The goal of this project is to demonstrate a publication-style statistical workflow using SAS and R, including data preparation, descriptive analysis, regression modeling, threshold exploration, and interpretation of immune biomarker patterns.

## Data Note

The original analysis used human cohort biomarker data.

Because the original dataset contains research data that may not be publicly shareable, this GitHub project will use documentation, annotated code, exported outputs, and simulated or summary-level data where appropriate.

## Planned Analysis Steps

1. Prepare and clean cohort-level data.
2. Define vitamin D exposure variables.
3. Summarize demographic and clinical characteristics.
4. Examine immune biomarker distributions.
5. Run adjusted regression models.
6. Explore nonlinear or threshold relationships.
7. Export SAS outputs as HTML reports.
8. Create visual summaries and interpretation notes.

## Tools

- R for selected visualizations and figure reconstruction
